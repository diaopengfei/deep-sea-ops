## 推荐阅读路径

- **新加入的开发**: 架构设计 → 后端代码导读 → Raft原理详解
- **运维人员**: 架构设计(第一节看目标、第五节看技术栈) → 部署指南
- **想深入分布式**: 架构设计(第四节) → Raft原理详解 → 后端代码导读(store/fsm 部分)

## 当前阶段

**v0.4 已完成**。单节点控制面、3 节点容错集群、Agent 骨架、登录鉴权、三路配置比对、Agent 自动扫描与生效配置合并、扩容迁移编排、G6 拓扑可视化、SSH 自动注入部署、入口代理、深度代码审查全部就绪。

### 版本概览

| 版本 | 核心内容 | 状态 |
|------|---------|------|
| v0.1 | 单节点控制面 + Agent 骨架 (Raft + bbolt + gRPC 双向流) | ✅ |
| v0.2 | 3 节点 Raft 容错集群 (动态加节点 + 故障切换) | ✅ |
| v0.3 | Java 运维 MVP + 安全鉴权 (登录/配置比对/自动扫描/扩容迁移/拓扑可视化) | ✅ |
| v0.4 | 自动部署 + 入口代理 (SSH 注入 + 凭据加密 + Leader 代理 + 代码审查) | ✅ |
